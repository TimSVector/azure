Basic Test Case Construction

Basic test case construction is covered, including input and expected values for scalar and structure types. The test cases that you build as part of the Interactive Tutorials are named (CL)MANAGER::PLACEORDER.001 and (CL)MANAGER::PLACE_ORDER.002.

For each test case, a Class Instance Object is created for Manager using the default contructor, and then the method under test is called.
